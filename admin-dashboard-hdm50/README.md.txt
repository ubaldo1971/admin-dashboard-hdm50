# HDM50 Dashboard

Este proyecto es el panel administrativo para gestionar los sorteos y pagos de autofinanciamiento de vehículos eléctricos Dolphin Mini en Hermosillo.

## Tecnologías
- Next.js
- React
- Tailwind CSS
- Vercel

## Funcionalidades
- Registro de sorteos mensuales.
- Registro y consulta de pagos por dirección de wallet.
- Conexión a un backend Node.js en MongoDB.
